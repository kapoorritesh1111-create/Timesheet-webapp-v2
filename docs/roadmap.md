# Product Roadmap

## Milestone 1 — Production Hardening (Security + Reproducibility)
**Exit criteria**
- Server-enforced route protection (middleware / server checks)
- Supabase schema is reproducible via migrations committed to repo
- Audit log exists for privileged actions (invites, approvals, rate changes, membership)
- Abuse protection on admin endpoints (rate limiting / checks)

## Milestone 2 — People Management v2
**Exit criteria**
- People directory (search, filters, status/role tags)
- Profile drawer/detail with tabs (Overview, Projects, Pay, Activity)
- Manager assignment and project assignment UX is consistent and safe

## Milestone 3 — Timesheet UX + Validation
**Exit criteria**
- Copy previous week
- Overlap + required-field validations
- Clear states (draft/submitted/approved/rejected) with explanations

## Milestone 4 — Approvals v2
**Exit criteria**
- Reject reason/comments
- Approval history per timesheet (audit view)
- Bulk approvals + exceptions view

## Milestone 5 — Payroll Runs
**Exit criteria**
- Pay periods + pay runs + frozen snapshots
- Exports (CSV summary/detail)
- Lock/unlock runs with audit trail
